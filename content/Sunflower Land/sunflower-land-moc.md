## 🌱 Sunflower Land là gì?

**Sunflower Land** là Web3 farming game trên blockchain (FLOWER ERC-20 trên BASE). Bạn trồng cây, nấu ăn, khai thác, câu cá, nuôi thú — rồi bán/trade tài sản ảo. 600,000+ farms đang hoạt động.

**Core Loop:**
```
Trồng cây → Thu hoạch → Nấu ăn → Ăn (gain XP) → Level up
    ↓                                            ↓
 Mua hạt giống ← Bán crop (Coins)         Mở khóa features mới
    ↓                                            ↓
 Mở rộng đảo → Prestige lên đảo lớn hơn → Endgame content
```

**4 loại tiền tệ:**
| Token | Vai trò | Kiếm bằng |
|-------|---------|-----------|
| 🪙 **Coins** | Mua seeds, vật liệu cơ bản | Bán crop, deliveries |
| 💎 **Gems** | Premium (restock shop, skip) | Mua bằng tiền thật/FLOWER |
| 🌸 **$FLOWER** | Crypto token (rút ra bán được) | Deliveries, staking |
| 🎫 **Chapter Tickets** | Event currency (hết hạn mỗi chapter) | Daily, bounties, chores |

→ [[tokens]] · [[about-sunflower-land]]
