**Status:** In development — beta estimated July/August 2026
**Official docs:** https://docs.sunflower-land.com/roadmap/project-ii

## What Is It

- **New game** in the FLOWER ecosystem (not another farming game)
- Click-based, browser-friendly
- **Monster collection** game
- Powered by **$FLOWER token**

## Key Details

- Genre: Trading + Web3 ownership, click-based
- Starting monster: **Grassling**
- Goal: Capture creatures, reach Firebug Level 2 (boss)
- Difficulty: Levels 1→60 (currently 1→20 in prototype)
- **Monster training** feature
- Monster stat generation: **fully random** with genetics and training value
- Own creatures (separate from Sunflower Land pets/buds)
- Mobile version is a priority

## FLOWER Token Role

- Native token for NFT drops, gameplay perks, exclusive content
- FLOWER primarily generated through Sunflower Land
- Play Sunflower Land → earn FLOWER → spend in Project II

## Timeline

- GitHub open: After core mechanics locked
- Beta: July/August 2026
- Full release: 2026
